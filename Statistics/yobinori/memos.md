# 点推定
ピンポイントの予想=点推定

目的標本から以下を推定する
- 母平均
- 母分散

平均の点推定
- 標本平均
  - 標本の平均をとれば母平均に近いであろうと考える
  - 標本平均も確率変数

分散の点推定
- 標本分散
  - 確率変数
  - あまり点推定に適していない
- 不偏分散
  - 点推定に適している
  - 分散の分子をn-1で割る
  - n-1で割った方がより母分散に近い数値となる

推定量に望む性質(θ:母数-真の値, θ^:推定量)
a. 一致性
  - nが十分大きければθ^はθに限りなく近づく
  - 標本分散も不偏分散も一致性を持っている
  - 数学的に曖昧
    - なぜ確率変数なのに限りなく近づくのか?
    - 数学的に証明するには**確率収束**を用いて差であるεより大きくなる数値が0に近づく事を証明する
b. 不偏性
  - θ^の期待値がθになる
  - 標本分散は不偏性を満たさない
    - 標本分散は真の文さんを過小評価してしまうから
```
式:
E[S^2] = θ^2 - 1/n θ^2
の形で1/nθ^2分だけ母分散を過小評価してしまう。
```
  - 不偏分散は不偏性を満たす
```
式:
E[U^2] = θ^2
```
  - なぜ過小評価するか(直感的に)
    - point
      - 標本平均の偏差であるということ
      - 標本分散の分子は**標本平均**(過小評価につながる)だが本当は母平均を使うべきだが使えない
        - 標本平均からの分散を求めるため必ず母平均からずれ結果的母分散からずれる(過小評価につながる原因)

# 推定/検定入門(区間推定:分散が既知な場合)
区間推定(母分散既知)
- 幅を持たせた推定
- 母平均は95%(1.96σ)の確率でこの区間に含まれることを推定する
- μ-1.96σ/√n <= x <= μ+1.96σ/√n = x-1.96σ/√n <= μ <= x+1.96σ/√n->信頼度95%の信頼区間
  - xは標本平均(平均μ分散σ^2/nの正規分布に従う)
    - 標本取れば取るほど母集団に近づく

信頼区間の意味
- 算出される信頼区間のうち95%が母平均をとる
- 標本取れば取るほど信頼区間が狭まる
- 一般式
  - X - k σ/√n <= μ <= X + k σ/√n
  - k = 標準正規分布の両側100(1-α)%点
    - αが信頼区間
    - 1.96とは分散1の標準正規分布の両側5%点を指す
      - 1 + (1 * 0.95) = 1.96
    - 　両側5%は片側2.5%づつのことを指す

# 区間推定/区間(区間推定-分散未知)
確率変数の標準化(平均0分散1)
- E[aX+b] = aE[X] + b
- V[aX+b] = a^2V[X]
上記性質を満たす

Z = (X - μ) / (σ/√n)は標準正規分布に従う

ここまでは母分散がわかっている状態

母分散が不明な場合
T = (X - μ) / (U / √n)
- U:不偏分散

ただこれは確率変数で確率変数を割っているために正規分布に従うとは言えない

この場合自由度n-1のt分布に従うことになる

t分布は標準正規分布より裾に広い分布となる

特徴:
- 左右対称
- 正規分布をつぶしたような分布
- nが大きくなると正規分布に近づく
- 自由度によってt分布の95%範囲が変化する

一般式

一般に正規母集団に対する信頼度αの信頼区間は

X - k (U/√n) <= μ　<= X + k (U/√n)

となりここでkは自由度n-1のt分布の両側100(1-α)%に従う

(nの数に寄って範囲が変化する)

# 区間推定/区間(区間推定-母集団分布が未知の場合
正規分布が仮定できない(未知)のときポイント
- 大きい標本が必要
- 不偏分散で母分散代用

中心極限定理に乗っ取る(大きい標本)と仮定
```
平均μ、分散σ^2の母集団から抽出した標本の大きさnが十分大きいとき
標本平均Xは近似的に平均μ、分散σ^2/nの**正規分布**に従う
```
α=5%としX-1.96(σ/√n) <= μ <= X + 1.96 (σ/√n)
これは正規分布と仮定した場合のなので未知の分布でnが十分大きい場合はσ->Uにそのまま置き換える
X-1.96(U/√n) <= μ <= X + 1.96 (U/√n)と考えることができる

# 推定検定(母比率の推定)
母比率
- ある特定の値が母集団に含まれる比率

標本比率
- ある特定の値が標本に含まれる比率

標本比率から母比率を推定していく

X = 1, 0

母比率の信頼区間
```
各X(i=1,2...n)はベルヌーイ分布
に従う(平均p, 分散p(1-p))
```

nが十分大きいとき、中心極限定理より(0,1でとるときはこの平均が母比率と考えて良い)

X = (X1+...Xn) / n = R

は、平均p分散(p(1-p))/nの正規分布に従う

信頼区間

R - 1.96 √p((1-p)/n) <= p <= R + 1.96 √p((1-p)/n)

値が不明なpが区間推定に用いられる詰んだ..

√(p(1-p)/n) を √(R(1-R)/n)に置き換える

->大数の法則によりpはRに限りなく近くなる

大数の法則

```
独立で同じ分布からなる確率変数の標本平均は母平均に収束する
```

R - 1.96 √R((1-R)/n) <= p <= R + 1.96 √R((1-R)/n)

これに置き換わる

二次不等式を活用してpをRに置き換えずに信頼区間を得ることができる
nが十分大きいときは計算精度はあまり変わらない

# 推定検定入門(母分散の推定)
設定
- 正規分布
- 分散を推定したい

できると何がよい
```
母集団で起きるであろう散らばりが推定できる
```

標本から作れる分散の分布が必要

定理
```
分散σ^2の正規分布に従う独立な確率変数X1,X2,...Xnがあるとき
T = ((n-1)U^2) / σ^2
は**自由度n-1のχ^2分布**に従う
```
U:不偏分散->従う分布はχ^2分布

[χ^2分布](https://ja.wikipedia.org/wiki/%E3%82%AB%E3%82%A4%E4%BA%8C%E4%B9%97%E5%88%86%E5%B8%83)

特徴
- 左右非対称
- 自由度に寄って大きく形状が変わる

χ^2分布の気持ち

T = ((X1-X)/σ)^2 + ((X2-X)/σ)^2 + ((Xn-X)/σ)^2 = (n-1) / σ^2 * ((X1-X)^2 + (X2-X)^2 + ... + (Xn-X)^2) / (n-1) = ((n-1) U^2) / σ^2

T = ((n-1) U^2) / σ^2
平均からのズレ度合い

ex:
n=10

自由度9のχ^2分布 95%の信頼区間
下側2.5%=2.7, 上側2.5%=19.0

((n-1)U^2) / 19 <= σ^2 <= ((n-1)U^2) / 2.7

これが信頼区間となる


# 推定検定入門(母平均の検定)
統計的検定
- 母集団に関する仮説を標本から得た情報に基づき検証すること

使う数学的手法
- 推定と同じ

使う論証手法
- 確率特有な断定しない論証となる

帰無仮説H0(相手の主張)
- 棄却したい仮説
- まず認める
- μの分布を調べる
- H1に有利な棄却域を設定する(有意水準)
  - 片側検定
    - 有意水準5%を取ると片側(左or右側)での5%が棄却域となる
  - 両側検定
    - 有意水準5%を取ると両側2.5%づつが棄却域となる
- 棄却域に標本平均が入ったら対立仮説を採択
- 棄却域に入らない場合は帰無仮説を受容する(棄却できない)

対立仮説H1(自分の主張)
- 採択したい仮説

注意
```
同じ帰無仮説でも対立仮説のとり方で逆の結果が得られることがある
例えば有意水準の設定方法が片側と両側で結果が変わるし%でももちろん結果が変わる
```

検定の流れ
1 H0とH1を設定
2 H0のもと対象となる統計量の分布を調べる
3 有意水準(危険率)を決め調べた分布に追いてH1に有利となる棄却域を設定する
4 標本を抽出し、統計量が棄却域にあるか調べ棄却域にあればH0を棄却する

危険率=有意水準
```
誤った結論を出す確率(危険を犯す)
危険率を下げればよりこの確率が下がるという考えのもとこの名前がついている
```
